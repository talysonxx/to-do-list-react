# ToDoList feito em Reactjs!

### Clone o projeto

### Na raiz, digite no terminal

#### `npm i`

Instala as dependências

### Depois, rode o projeto com

#### `npm start`

Irá abrir uma guia na URL [http://localhost:3000/](https://localhost:3000)
